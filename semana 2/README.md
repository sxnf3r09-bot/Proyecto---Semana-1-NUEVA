# Proyecto Semana 2 – Ficha de Datos del Dominio

## Dominio
Servicios Financieros Digitales (Fintech)

## Descripción
Este proyecto consiste en crear un script en JavaScript que se ejecuta con Node.js y muestra en la consola una ficha de datos sobre una entidad del dominio asignado.

En este caso el dominio es **Fintech** y la entidad utilizada es una **aplicación de ahorro programado**.

El programa muestra información técnica de la app como su nombre, código de producto, meta de ahorro y estado activo, además de verificar los tipos de datos de las variables y realizar una conversión de tipos.

## Objetivo del proyecto
Practicar los conceptos aprendidos en la semana 2:
* Uso de variables con `const`
* Tipos de datos en JavaScript
* Uso de `typeof`
* Conversiones de datos
* Uso de `console.log()`
* Uso de template literals
