# Proyecto Semana 1 – Modelado de Clases y Objetos

## Descripción
Este proyecto consiste en crear un script en JavaScript que muestra en la consola una ficha de presentación técnica de una entidad.

La entidad principal es un producto financiero digital diseñado para la gestión de ahorros y transacciones de usuarios finales. El programa utiliza clases y métodos para organizar la información y se ejecuta mediante Node.js.

## Objetivo
Aplicar los conceptos fundamentales de programación aprendidos en la primera semana:
* Uso de `console.log()` para salida de datos.
* Manipulación de tipos de datos primitivos (Strings, Numbers, Booleans).
* Implementación de expresiones aritméticas (Cálculo de costo neto).
* Uso de plantillas de cadena (Template literals).
* Organización de código y comentarios.
